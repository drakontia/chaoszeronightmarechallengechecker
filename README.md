# ChaosZeroNightmareChallengeChecker

ゲーム「カオスゼロナイトメア」のシーズン挑戦課題をチェックし、LocalStorage に保存できる攻略補助サイトです。

## 技術スタック

- Next.js 16 (App Router)
- Tailwind CSS 4
- TypeScript 6
- next-intl（日本語メッセージファイル経由）
- Vitest / Playwright
- pnpm

## 開発コマンド

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
pnpm test:unit
pnpm test:e2e
```
